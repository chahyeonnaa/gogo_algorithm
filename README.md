## 규칙 :cop:
1. 이슈발급- 문제에 대한 설명, 알고리즘, 어려웠던 부분, 느낀점 등을 상세히 기록  
   (이슈 제목 : [프로그래머스/백준 등] 문제 제목)  
   _(ex) [프로그래머스] 신규아이디 추천_  
  

2. 각자의 브랜치에 커밋 후 푸쉬  
   (커밋규칙: "문제 이름/(이슈번호)")  
   _(ex) "신규아이디 추천/(#2)"_
  

3. 팀원들의 이슈에 승인  
   (마스터 브랜치로 병합은 모두가 승인한 후 본인이 진행)
#### * 일주일에 최소한 한 문제는 풀어서 올리는 것을 원칙으로 한다
#### * 문제는 개인이 선정한다(언어도 자유롭게 선택)
